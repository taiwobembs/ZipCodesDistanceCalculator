### Getting Started

1. php artisan migrate
2. php artisan db:seed
3. php artisan serve
4. vendor/bin/phpunit


### Requirements

- PHP, NPM , MYSQL

**Please note:**

- Change base url in FRONTEND store as needed, pay attention to the port
